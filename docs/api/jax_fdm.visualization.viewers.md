# Desktop 3D

An interactive desktop 3D viewer built on
[compas_viewer](https://github.com/compas-dev/compas_viewer) with a scene tree, element selection, and animation support.

!!! note

    This viewer needs optional dependencies included in the `viz`
    extra. Without them, the viewer degrades to a null object that warns on use.

::: jax_fdm.visualization.viewers.viewer.Viewer
    options:
      heading_level: 2
