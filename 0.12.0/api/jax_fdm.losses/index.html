    Losses¤

    A loss function scalarizes the deviation between the current equilibrium state and the goals into the objective that an optimizer minimizes.

    Loss¤

    losses ¤

    A function composed of error and regularization terms.

    The regularization terms in the loss function.

    PredictionError ¤

    PredictionError(goals, alpha=1.0, name=None, *args, **kwargs)

    The prediction error.

    You lose when you get too much of something.

    MeanPredictionError ¤

    MeanPredictionError(goals, alpha=1.0, name=None, *args, **kwargs)

    The mean prediction error.

    Average out all errors because no single error is important enough.

    SquaredError ¤

    SquaredError(goals, alpha=1.0, name=None, *args, **kwargs)

    The canonical squared error.

    It measures the L2 distance between the current and the target value of a goal.

    MeanSquaredError ¤

    MeanSquaredError(goals, alpha=1.0, name=None, *args, **kwargs)

    The seminal mean squared error.

    Average out all errors because no single error is important enough.

    RootMeanSquaredError ¤

    RootMeanSquaredError(goals, alpha=1.0, name=None, *args, **kwargs)

    The root mean squared error.

    AbsoluteError ¤

    AbsoluteError(goals, alpha=1.0, name=None, *args, **kwargs)

    The canonical absolute error.

    It measures the absolute difference between the current and the target value of a goal.

    MeanAbsoluteError ¤

    MeanAbsoluteError(goals, alpha=1.0, name=None, *args, **kwargs)

    The canonical mean absolute error.

    LogMaxError ¤

    LogMaxError(goals, alpha=1.0, name=None, *args, **kwargs)

    The log error for constraints with a target maximum value that should not be exceeded.

    Helpful to deal with soft barrier constraints with an upper bound.
