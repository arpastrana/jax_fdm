# Jupyter notebooks

A draw-once pythreejs widget for Jupyter notebooks and Colab, built on
[compas_notebook](https://github.com/compas-dev/compas_notebook).

!!! note

    This viewer needs optional dependencies included in the `viz`
    extra. Without them, the viewer degrades to a null object that warns on use.

## Notebook Viewer

::: jax_fdm.visualization.notebooks.viewer.NotebookViewer
    options:
      heading_level: 2
