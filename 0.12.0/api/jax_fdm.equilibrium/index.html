    Equilibrium¤

    The heart of JAX FDM: compute states of static equilibrium with the force density method, differentiably.

    Form-finding¤

    The two entry points. fdm computes an equilibrium state for fixed parameters; constrained_fdm searches for the parameters whose equilibrium state minimizes a loss function, subject to constraints.

    Models¤

    EquilibriumModel ¤

    EquilibriumModel(tmax=100, eta=1e-06, is_load_local=False, itersolve_fn=None, iterload_fn=None, implicit_diff=True, verbose=False)

    A FDM model to calculate equilibrium states with shape-dependent loads.

    Parameters:

    Name Type Description Default
    tmax `int`

    The maximum number of iterations to calculate an equilibrium state. If tmax=1, the model is equivalent to doing one linear FDM step, and the rest of the parameters of this model are ignored. The edge and face loads are discarded too. Defaults to 100.

    		 100
    eta `float`

    The convergence tolerance for calculating an equilibrium state. Defaults to 1e-6.

    		 1e-06
    is_load_local `bool`

    If set to True, the face and edge loads are applied in their local coordinate system at every iteration (follower loads). Defaults to False.

    		 False
    itersolve_fn `Callable`

    The function that calculates an equilibrium state iteratively. If None, the model defaults to forward fixed-point iteration. Note that the solver must be consistent with the choice of residual function. Defaults to None.

    		 None
    iterload_fn `Callable`

    A load callback that is invoked before starting iterative equilibrium computation. Defaults to None.

    		 None
    implicit_diff `bool`

    If set to True, it applies implicit differentiation to speed up backpropagation. Defaults to True.

    		 True
    verbose `bool`

    Whether to print out calculation info to the terminal. Defaults to False.

    		 False
